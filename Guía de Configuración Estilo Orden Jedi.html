🌌 Guía de Configuración Estilo Orden Jedi

1️⃣ Fondo de Pantalla con i3

sudo pacman -S wget mkdir -p ~/Pictures wget -O ~/Pictures/nombre_que_le_imagen.jpg https://ruta-de-la-imagen.jpg sudo pacman -S feh feh --bg-fill ~/Picture/fondo_venator.jpg

Edita ~/.config/i3/config y agrega:

exec_always --no-startup-id feh --bg-fill ~/Picture/fondo_venator.jpg

🚀 Instalar Brave con yay

yay -S brave-bin Brave & disown

🧠 Instalar Starship

yay -S starship

Agrega en ~/.bashrc:

eval "$(starship init bash)" source ~/.bashrc

😄 Emoji Fonts

sudo pacman -S noto-fonts-emoji

mkdir -p ~/.config nano ~/.config/starship.toml
